<b>═ 🔥 TOP STORY ═</b>
Espressif formally unveiled the ESP32-S31, the direct architectural successor to the ESP32-S3, replacing the Xtensa LX7 dual-core with a dual-core RISC-V at 320 MHz that adds a 128-bit wide data path with SIMD instructions and an MMU, alongside an integrated Gigabit Ethernet MAC — the first time wired Ethernet has returned to the flagship ESP32 line. The SoC combines Wi-Fi 6 (2.4 GHz 802.11ax), Bluetooth 5.4 Classic/LE, and IEEE 802.15.4 (Thread/Zigbee/Matter) in a single die, alongside RAM-based PUF key management, hardware secure boot, and a Trusted Execution Environment with Access Permission Management. For ESP32-S3-based designs like Life_logger, this chip defines the hardware target for the next firmware generation: one SoC covers BLE, Thread, Wi-Fi 6, and wired Ethernet with a hardware TEE baked in. Samples are reaching select OEM accounts; no public dev-board date has been announced and ESP-IDF support is confirmed but not yet in a stable release branch.
