<b>═ 🔥 TOP STORY ═</b>
Zephyr 4.4.0 ships WireGuard VPN as a first-class networking subsystem — the first production open-source RTOS to include it natively. Enabled via CONFIG_WIREGUARD, it uses Zephyr's standard socket API so existing UDP application code routes through authenticated encrypted tunnels without architectural changes, making secure device-to-device and device-to-cloud communication a configuration choice rather than a hardware add-on. The same release adds Wi-Fi Direct (P2P) so Zephyr nodes can establish direct encrypted links without an access point. Combined, these features make Zephyr 4.4 the most connectivity-capable RTOS version to date, and they land at exactly the moment the EU CRA begins mandating authenticated firmware update and secure communication channels for connected embedded products. For any new IoT firmware design targeting the EU market, Zephyr 4.4 now closes the gap between "bare-metal RTOS" and "secure-by-default network stack" without requiring a full embedded Linux stack.
